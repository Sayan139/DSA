## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/DSA.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd DSA
   ```

3. **Compile and run a file:**
   ```bash
   g++ -o output FileName.cpp
   ./output
   ```
